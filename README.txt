# TRC Education Website

Free static website for GitHub Pages.

## Files
- `index.html`
- `style.css`
- `trc-education-logo.png`

## Recommended GitHub repository
Create a public repository named:

`trcedu`

Then upload these files.

## GitHub Pages
Repository → Settings → Pages → Deploy from branch → main → root.

## Custom domain
In GitHub Pages custom domain field, enter:

`www.trcedu.com`
